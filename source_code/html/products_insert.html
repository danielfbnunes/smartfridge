What can you do in this section?


Insert Products Using Bar Codes

Pass the product's bar code on the bar code reader to insert it in the fridge. If you pass it several times, you will be increasing its quantity. Some products might not exist in the bar code catalog.


Insert Products Manually

If you a product that doens't have a bar code, you can insert it manually. Just write the name and the quantity of the product and you are set to go.


Edit Products

It's possible to edit the expiration date of a product and add some comments regarding it.
